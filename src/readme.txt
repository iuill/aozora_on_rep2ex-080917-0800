rep2で青空文庫形式のテキストファイルを読めるようにしてみました。
使う場合は自己責任でよろしく。

同梱のrep2フォルダには差分があるファイルだけ入れてあります。
拡張パック入れてることが前提なので、拡張パック入れてない人は
まずはこっから落としてください。

  rep2-expack: http://page2.xrea.jp/expack/


画像を表示する場合は、AASを有効にしてください。


青空文庫形式のテキストファイルの置き場所は、
データ保存ディレクトリ配下に下記のような形で
ファイルを配置してください。

    データ保存ディレクトリ\p2_aozora\text\任意のフォルダ名\任意のテキストファイル名(青空文庫形式のテキストファイル)



それなりには読めるようになっていると思いますが、
PCではルビを表示させるとレイアウトが崩れます。
その他、個人的に優先度低いものは実装していません。
